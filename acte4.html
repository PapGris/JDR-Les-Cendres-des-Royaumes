Acte IV : Le Combat Final au Volcan

Une bataille dans les flammes

Le culte du feu atteint le sommet du Mont Noir pour invoquer un démon du magma. Les aventuriers doivent les arrêter avant qu’ils ne déclenchent une éruption cataclysmique.

Défis :

  • Des vagues d’ennemis : élémentaires de feu, golems de lave, et des cultistes fanatiques.
  • Un combat final contre le Grand Prêtre Pyros et le Démon de Brasier.

Choix final :

Une fois le démon confronté, les aventuriers doivent prendre une décision cruciale concernant l'Œil de Cendre.

Détruire l’Œil de Cendre pour bannir le démon :

  • Conséquences positives : Le démon est vaincu, et le volcan est scellé pour des siècles.
  • Conséquences négatives : L’artefact est perdu, privant le monde de sa protection.

Utiliser l’Œil pour contrôler le démon :

  • Conséquences positives : Les aventuriers peuvent utiliser le démon comme arme puissante.
  • Conséquences négatives : Le volcan reste actif, mettant la région en péril.

Ne rien faire :

  • Conséquences négatives : Le démon est libéré, déclenchant un âge de destruction.

Épilogue : L’Héritage des Aventuriers

Les décisions des joueurs affectent la région pour les générations à venir :

  • Le village gobelin, la forêt, et le marais prospèrent ou sombrent dans le chaos selon les alliances.
  • Le Mont Noir est soit apaisé, soit une menace permanente.
  • Les aventuriers deviennent des héros, des figures controversées, ou des parias, selon leurs choix.
